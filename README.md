# Chord Detector

Web aplikacija za automatsku detekciju glazbenih akorada iz audio zapisa.

## Pokretanje

### 1. Instaliraj dependencies
pip install -r requirements.txt

### 2. Pokreni aplikaciju
python app.py

### 3. Otvori u browseru
http://127.0.0.1:5000

## Korištenje
- Uploadaj WAV ili MP3 file
- Ili snimi audio direktno u aplikaciji
- Klikni "Detect Chords"
- Rezultati se prikazuju u tabličnom ili live modu
